# 🌱 Farmer World — Organic Marketplace UI

A modern, responsive frontend application for an organic farming marketplace.
It showcases seeds, fruits, and vegetables with a clean UI, cart functionality, and smooth user experience.

---

## 🚀 Live Demo

👉 https://farmer-world-ten.vercel.app

---

## 📌 Features

* 🛒 Add to Cart functionality
* 🔍 Product listing (Seeds, Fruits, Vegetables)
* 🌙 Dark mode support
* 📱 Fully responsive (mobile-first design)
* ⚡ Fast performance (Vite + optimized assets)
* 🎯 Clean UI with modern design system

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/amansharma073/farmer-world.git
cd farmer-world-app
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
